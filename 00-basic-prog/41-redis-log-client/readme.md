*how to use it*

1. compile simple redis client example
2. install redis locally like: `sudo apt-get install redis`
3. start redis service like: `sudo systemctl start redis`
4. run ./00-basic-prog/41-redis-log-client/Debug/41-redis-log-client
5. you should see all connected and written to redis
6. check redis with: `redis-cli` && `XRANGE leo_phone_11_03_2026 - +`
7. you should see all logs in output
